# felicitys-personal-site

## About This Project

This is my personal website. It is a simple but fun site where visitors can learn a little about me, explore my projects, and try a few interactive buttons. I made it while practicing HTML, CSS, and JavaScript, and it is definitely an improvement from my last website.

## Features

- Multiple pages, including Home, About Me, Projects, and a custom 404 page
- A cute pastel theme
- Fun animated GIFs
- Interactive buttons for changing the background color, text color, font, and dark mode
- Responsive images that work better on smaller screens
- A small hover effect on images

## Languages Used

- HTML
- CSS
- JavaScript

## Project Files

- `index.html` is the main home page.
- `about.html` introduces me.
- `projects.html` links to my projects.
- `style.css` controls the design, colors, layout, and animations.
- `script.js` controls the interactive buttons.
